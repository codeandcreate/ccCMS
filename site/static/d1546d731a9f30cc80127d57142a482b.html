Just a quick help on what you see...

Each .html url references to a static file in /site/static/. The url will be md5 hashed, so "/index.html" (or just /) gives d1546d731a9f30cc80127d57142a482b. The file you see is /site/static/d1546d731a9f30cc80127d57142a482b.html.

To a dynamic example, located in /sites/dynamic/example.php, configured in /sites/ccCMS_dynamic.conf.php.
